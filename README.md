## Портфолио проектов

Все проекты представлены в отдельных репозиториях на GitHub. К каждому приложено описание, инструкция по запуску и скриншоты.

**Адаптивные лендинги.** Чистая верстка на HTML и CSS с использованием Flexbox, Grid и анимаций. Проекты корректно отображаются на десктопах, планшетах и мобильных устройствах. 

RAYWAKE — лендинг умного будильника. Полностью адаптивная и кроссбраузерная верстка на HTML5 и CSS3 (Flexbox, Grid, CSS-анимации). Реализована анимация цифр при скролле через Intersection Observer API, автоматическая карусель преимуществ, мобильное меню, маска ввода телефона и клиентская валидация формы с имитацией отправки. Чистый JavaScript без фреймворков. Ссылка на лендинг: https://kaymka.github.io/RAYWAKE/

NERO — промо-лендинг уличного бренда в киберпанк-эстетике. Адаптивная верстка на HTML5 и CSS3 (Flexbox, Grid, CSS-анимации). Реализованы кастомный курсор, горизонтальный скролл коллекций, глитч-эффект текста, бегущая строка, параллакс изображения, анимация появления элементов при скролле (Intersection Observer API) и неоновый котик-компаньон, прыгающий к интерактивным блокам. Форма подписки с валидацией email. Мобильное меню с бургер-кнопкой. Весь JavaScript — чистый, без фреймворков. Ссылка на лендинг: https://kaymka.github.io/Nero/

NEBULA STUDIO — лендинг digital-студии в футуристическом стиле. Адаптивная верстка на HTML5 и CSS3 (Flexbox, Grid, CSS-анимации, кастомные формы). Реализованы анимированный градиентный фон с парящими сферами, морфинг геометрических фигур, плавающие иконки, кастомный курсор с эффектами наведения, горизонтальный скролл для процесса работы с собственной стилизацией, мозаичная сетка портфолио с оверлеями при ховере, фильтрация проектов, карусель отзывов, сплит-экран для контактов с формой и валидацией полей, а также анимации появления элементов при скролле через Intersection Observer API. Мобильное меню с бургер-кнопкой и плавным выезжанием, кнопка «Наверх». Весь JavaScript — чистый, без фреймворков. Ссылка на лендинг: https://kaymka.github.io/NEBULA/

**Приложения на TypeScript и JavaScript.** Я написал несколько рабочих веб-приложений: планировщик задач (to-do лист) с сохранением состояния, виджет погоды с запросами к реальному API, калькулятор с историей вычислений. В этих проектах я использовал DOM-манипуляции, асинхронные запросы, обработку ошибок и чистую архитектуру.

**Веб-приложения на PHP.** Форма обратной связи с валидацией и отправкой на email, простая система авторизации с сессиями, мини-блог с подключением к базе данных SQL. В этих проектах реализована работа с POST-запросами, защита от базовых уязвимостей и структурирование кода.

**Проекты с Bootstrap.** Несколько сайтов, где я использовал Bootstrap для ускорения верстки, сохраняя при этом кастомную стилизацию и адаптивность.

Все проекты написаны мной самостоятельно в рамках практической разработки. Код чистый, прокомментирован в сложных местах и готов к ревью.
